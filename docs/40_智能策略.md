# 智能策略

## 概述

智能策略是 EMQuant 量化终端提供的自动化交易辅助功能，包括条件单、网格交易等策略工具。

> **注意**: 本页内容为占位摘要，完整的智能策略文档请参考 EMQuant 量化终端内置帮助或访问 [在线文档](https://emquant.18.cn/help/?doc=smart-strategy)。

## 条件单

条件单允许用户设定触发条件（如价格突破、涨跌幅等），当市场满足条件时自动执行买入或卖出操作。

### 主要功能

- 价格条件触发（高于/低于指定价格）
- 涨跌幅条件触发
- 成交量条件触发
- 多条件组合（与/或逻辑）

## 网格交易

网格交易策略通过预设价格区间和网格密度，在价格波动中自动进行高抛低吸操作。

### 主要参数

- 价格区间上下限
- 网格数量 / 网格间距
- 每格交易数量
- 触发方式
